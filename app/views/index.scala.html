Dotto

In 2016 I decided that Scala was going to be the new language I was going to learn. In order to start playing with Scala I decided to develop a pet project to translate plain text messages into the morse representation. Using a led connected at a RaspberryPi showing the translated message as led pulses. Dotto is an Open Source project created to learn Scala. Visit https://www.github.com/Dotto if you want to contribute.
